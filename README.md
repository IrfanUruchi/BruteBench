# BruteBench
 BruteBench is a lightweight system benchmarking tool designed to evaluate real-world performance for development workloads.The tool focuses on practical measurements instead of synthetic benchmarks, combining CPU, memory, system, and GPU workloads into a unified scoring model.


# Launching version 1 by Tuesday 31st March 2026

---

## Purpose

The goal of BruteBench is to provide a clear and realistic view of how a system behaves under conditions similar to everyday engineering tasks such as:

- system programming
- data processing
- AI/ML workloads
- backend development

---

## Features

- CPU benchmark (operations per second, stability)
- Development workload simulation (multi-round execution)
- Memory pressure testing with latency tracking
- System-level mixed workload evaluation
- GPU benchmarking:
  - Apple Silicon (MLX)
  - NVIDIA CUDA
  - CPU fallback
- Use-case based scoring:
  - Mobile Development
  - Backend Development
  - Systems Engineering
  - Data / Pipelines
  - DevOps
  - AI / Machine Learning
